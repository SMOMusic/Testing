Zero-click info: Replit
Replit, formerly Repl.it, is a San Francisco-based start-up and an online integrated development environment. Replit being Software as a service allows users to create online projects and write code in many supported languages. Its features include collaborative coding, syntax highlighting, code completion, debugging, hosting, unit testing, version control, package management, code generation via artificial intelligence, and more.
